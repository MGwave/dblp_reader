This folder contains the following files:
1. dblpTotalType.txt
	Each line represents an entity of DBLP. Each line starts with the id, following by a list of type ids.
	
2. dblpAdj.txt
	Each line represents an edge of DBLP, with format "sid	eid	edgeType".
	
3. dblpType.txt
	Each line matches an edge type to its name.
	
4. author_label_new.txt
	Each line matches an author to his/her area.
